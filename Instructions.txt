Repository:
https://github.com/meshchem/UrbanComputingApplication.git

1. clone repo
- git clone https://github.com/meshchem/UrbanComputingApplication.git
- cd UrbanComputingApplication

2. setup backend
- backend runs on Python (FastAPI)

- start virtual env
    - python3 -m venv venv
    - source venv/bin/activate
- Install dependencies:
    - pip install -r requirements.txt

3. Start the FastAPI backend
- open terminal
    - run: uvicorn main:app --reload

This runs the backend on :http://127.0.0.1:8000
Inspect: http://127.0.0.1:8000/docs

4. Ngrok to expose backend
(SensorLogger required public HTTPS endpoint)
Ngrok forwards the local FastAPI server to a secure URL.
- open another terminal
    - run: ngrok http 8000

Ngrok public HTTPS URL:  https://jaqueline-unhydraulic-anachronically.ngrok-free.dev

5. SensorLogger app
- Settings -> Data streaming:
    - enable HTTP Push
    - Push URL: https://jaqueline-unhydraulic-anachronically.ngrok-free.dev/sensor/{library}/{floor}

6. Frontend (React + TypeScript)
- open new terminal
    - cd tclib-frontend
    - npm install
    - npm run dev
